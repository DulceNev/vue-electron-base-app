# Base app con electron + vue

¡Bienvenido! 
Esta es una base para aplicaciones de escritorio construida con **Electron**, **Vue 3** y **Vite**, es mi base para simplemente clonar y no tener que crear el proyecto de 0 Está diseñada para ser ligera, rápida y fácil de personalizar.

## 🛠️ Tecnologías utilizadas

* **Framework:** [Vue 3](https://vuejs.org/) (Composition API)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Desktop Wrapper:** [Electron](https://www.electronjs.org/)

## 🎨 Stack de Diseño

* **Tailwind CSS:** Framework de utilidades para un control total del diseño sin salir del HTML.
* **daisyUI:** Componentes listos para usar (botones, modales, cards) basados en clases de Tailwind

Asegúrate de tener [Node.js](https://nodejs.org/) instalado. Luego, clona el repositorio e instala las dependencias: npm install
Para ejecutar la aplicación en modo desarrollo con recarga en tiempo real: npm run start
